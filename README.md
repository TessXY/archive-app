Archive App è un progetto Spring Boot progettato per archiviare e gestire foto, con un sistema di utenti, ruoli e autenticazione.
L’app utilizza PostgreSQL come database e Liquibase per la gestione delle migrazioni.

Il repository non contiene dati sensibili: i file di configurazione come application.yml sono volutamente esclusi per proteggere credenziali e impostazioni locali.

Tecnologie principali:
- Java 21
- Spring Boot 3.5
- Thymeleaf
- PostgreSQL
- Liquibase
